# Workflow scaffolding (apply at project start)

Canonical version of this content lives in the Skill:
`project-template/.claude/skills/workflow-scaffolding/SKILL.md`. Copy that Skill into new
projects — it loads on demand at project start instead of every session.

Keep this Rule file only if a project wants the scaffolding checklist always-active in
static context rather than loaded on demand (rare — most projects should prefer the Skill).
If you do keep it, copy the Skill's content here rather than duplicating a second,
independently-drifting copy.
